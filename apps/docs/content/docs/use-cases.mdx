## Multi-Step Onboarding

Problem: onboarding usually asks for several fields, validates answers, and branches based on user type. Plain handlers often spread that state across sessions and callbacks.

Fluxgram shape: one flow with text prompts, validation, buttons, and a final dynamic step that reads `ctx.store`.

```ts
fx.flow("onboard", [
  prompt.text("Name?", { store: "name" }),
  prompt.buttons("Plan?", { buttons: [btn("Team", set("plan", "team"))] }),
  (ctx) => send(`Welcome ${ctx.store.name}.`),
]);
```

## Support Triage

Problem: support triage needs category, urgency, screenshots, and contact details. Users may disappear and return later.

Fluxgram shape: combine buttons for structured choices, `prompt.message` for attachments, and validation for contact data. Support triage is a durable flow, not a pile of temporary session flags.

## Reminder And Follow-Up Bots

Problem: reminders and follow-ups need to pause for minutes, hours, or days without holding memory in the process.

Fluxgram shape: use `sleep`, `humanSleep`, or `waitFor`; sweeps resume due timers from durable storage.

```ts
fx.flow("follow-up", [send("I will check back later."), sleep(3600), send("Checking back in.")]);
```

## Group Administration

Problem: group bots need setup flows, admin-loss handling, migration tracking, and reset commands that do not strand old prompts.

Fluxgram shape: use group lifecycle hooks, `overrideActive` reset commands, and migration-aware active-flow inspection.

## Internal Operations Bots

Problem: cron jobs and backend services need to start flows, send messages, and avoid double-running retried work.

Fluxgram shape: publish work through `FluxgramClient`, use `uniqueKey` for idempotency, and use `oneAtATimeKey` when events for the same resource must serialize.

## Moderation And Review Tools

Problem: reviewers need source context, structured decisions, and an audit-friendly path through a case.

Fluxgram shape: forward source messages, collect reviewer choices with buttons, store decisions, pin relevant messages, and send debug digests to an operator chat.
